## 📅 Day 3: Logical Operators & Conditional Statements

### 🚀 Challenges Covered
* **Divisibility Checker:** Verifies if a given integer is simultaneously divisible by both 3 and 5.
* **ATM Withdrawal Simulator:** Validates transactions by ensuring the withdrawal amount is positive and does not exceed the available balance.
* **Admission Eligibility:** Determines if a student qualifies for admission based on academic performance (Marks ≥ 70) *or* possessing a sports quota certificate.
* **Leap Year Finder:** Evaluates whether a specific year is a leap year using standard calendar logic.

### 🛠️ Key Concepts Applied
* **Logical AND (`&&`):** Used when all conditions must evaluate to `true` (e.g., checking both divisibility rules).
* **Logical OR (`||`):** Used when at least one condition must evaluate to `true` (e.g., meeting either admission requirement).
* **Control Flow:** Utilizing `if`, `else if`, and `else` conditional branching.
* **Modulo Operator (`%`):** Used to check for clean division by evaluating the remainder.
